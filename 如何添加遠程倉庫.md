# 如何添加遠程倉庫 (Git Remote)

## 步驟 1：確認你的 GitHub 帳號和倉庫

1. 登入 GitHub：https://github.com
2. 確認你的 GitHub 帳號名稱（例如：`johnsmith`）
3. 確認倉庫名稱（應該是 `CS3343_Project`）

## 步驟 2：添加遠程倉庫

### 方法 A：使用 HTTPS（推薦，簡單）

在 PowerShell 中執行：

```powershell
cd "C:\Users\henry\Downloads\CS3343_Project-checkout-20251118(1)\CS3343_Project-checkout-20251118"

# 添加遠程倉庫（請替換 YOUR_GITHUB_USERNAME 為你的 GitHub 帳號）
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/CS3343_Project.git
```

### 方法 B：使用 SSH（如果你已經設置了 SSH key）

```powershell
git remote add origin git@github.com:YOUR_GITHUB_USERNAME/CS3343_Project.git
```

## 步驟 3：驗證遠程倉庫已添加

```powershell
git remote -v
```

應該會顯示：
```
origin  https://github.com/YOUR_GITHUB_USERNAME/CS3343_Project.git (fetch)
origin  https://github.com/YOUR_GITHUB_USERNAME/CS3343_Project.git (push)
```

## 步驟 4：推送代碼

```powershell
git push -u origin feature/check-vacancy-tests
```

## 常見問題

### Q: 如果遠程倉庫已經存在怎麼辦？

如果已經有 `origin` 遠程倉庫，你可以：

**選項 1：刪除舊的，添加新的**
```powershell
git remote remove origin
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/CS3343_Project.git
```

**選項 2：修改現有的遠程倉庫 URL**
```powershell
git remote set-url origin https://github.com/YOUR_GITHUB_USERNAME/CS3343_Project.git
```

### Q: 如何查看所有遠程倉庫？

```powershell
git remote -v
```

### Q: 如何刪除遠程倉庫？

```powershell
git remote remove origin
```

## 完整示例

假設你的 GitHub 帳號是 `henry123`：

```powershell
# 1. 進入專案目錄
cd "C:\Users\henry\Downloads\CS3343_Project-checkout-20251118(1)\CS3343_Project-checkout-20251118"

# 2. 添加遠程倉庫
git remote add origin https://github.com/henry123/CS3343_Project.git

# 3. 驗證

# 4. 推送代碼
git push -u origin feature/check-vacancy-tests
```

## 注意事項

⚠️ **重要**：在執行 `git remote add origin` 之前，請確保：
1. 你已經在 GitHub 上創建了 `CS3343_Project` 倉庫
2. 倉庫名稱完全正確（大小寫敏感）
3. 你有該倉庫的寫入權限
